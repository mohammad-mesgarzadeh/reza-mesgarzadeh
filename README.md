# Reza Mesgarzadeh

Personal portfolio website of Reza Mesgarzadeh, showcasing telecommunications services, PBX and VoIP solutions, engineering projects, and handcrafted telecom tower scale models.

## Features

* Telecommunications and PBX services presentation
* Telecom tower scale model showcase
* Responsive modern design
* Product detail pages
* Contact section
* Bilingual support (Persian / English)
* Optimized for desktop and mobile devices

## Technologies

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* Lucide Icons
